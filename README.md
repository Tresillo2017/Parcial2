# Parcial 2 - Gestión de Empleados

Proyecto de ejemplo para gestionar empleados.

Autor: tresillo2017
Versión: 1.0

Contenido:
- `Empleado.java` Clase que representa un empleado con validaciones.
- `Empleados.java` Clase que gestiona la lista de empleados y operaciones.
- `Cargo.java` Enumeración de cargos permitidos.
- `Textos.java` Constantes de texto y números.
- `SistemaGestionEmpleados.java` Aplicación principal (contiene `main`).
- `TESTING.md` Análisis de caja negra para la clase `Empleado`.

Ejecución:
1. Abrir una terminal integrada en VS Code.
2. Compilar: `javac *.java`
3. Ejecutar: `java SistemaGestionEmpleados`

Tests (opcional):
Se han incluido pruebas JUnit en `test/EmpleadoTest.java`. Para ejecutarlas sin un sistema de compilación (Maven/Gradle), descarga los JARs de JUnit 5 y su plataforma desde Maven Central y añade los JARs al classpath al ejecutar `java`.

Notas:
- Asegúrate de tener instalada la extensión Java para VS Code y las opciones de ejecución en terminal configuradas si `Scanner` no funciona.
- Las cadenas y números fijos se han centralizado en la clase `Textos`.
